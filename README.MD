## 📌 About

A futuristic, neon-themed personal portfolio built with pure **HTML, CSS, and JavaScript** — no frameworks, no build tools. Deployed on Vercel with live GitHub project integration.

B.Tech (CSE - Data Science) student at Vaagdevi Engineering College, JNTUH.

---

## ✨ Features

- ⚡ **Three.js animated hero** — interactive particle field with 3D geometric shapes
- 🔤 **Typing effect** — role carousel with smooth typewriter animation
- 🌗 **Light / Dark theme toggle**
- 📡 **Live GitHub API integration** — projects fetched and rendered dynamically
- 🖱️ **Custom cursor** with neon glow trail
- 🎞️ **Scroll-triggered reveal animations** throughout all sections
- 📱 **Fully responsive** — mobile, tablet, laptop, and large screens
- 🎨 **Neon cyberpunk aesthetic** using CSS variables and glassmorphism cards

## 🛠️ Tech Stack

| Layer | Technologies |
|-------|-------------|
| Structure | HTML5 |

| Interactivity | Vanilla JavaScript (ES6+) |
| 3D Graphics | Three.js (r128) |
| Fonts | Google Fonts — Orbitron, Rajdhani, Share Tech Mono |

## 🚀 Run Locally

No dependencies, no build step needed.

```bash
# Clone the repo
git clone https://github.com/nikhilsai0803/nikhilsai0803.git

# Open directly in browser
open index.html
```

Or use a local dev server (recommended for Three.js):

```bash
# Using VS Code Live Server extension, or:
npx serve .
```
